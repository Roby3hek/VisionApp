# VisionApp
# Проект был выполнен в качестве задания на сессии
Этот проект представляет собой веб-приложение для компьютерного зрения (Computer Vision) на основе OpenCV.js. Оно позволяет выполнять обработку изображений, видео и потока с камеры непосредственно в браузере без необходимости установки дополнительного программного обеспечения на стороне клиента.
